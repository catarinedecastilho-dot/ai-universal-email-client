# Development Workflow: Claude Code & Agentic Dev

I followed a **Specs-Driven Development** approach using Claude Code CLI. 
1. **Initial Spec**: Defined the `CLAUDE.md` to establish project boundaries.
2. **Component Generation**: Used Claude to build atomic React components for account switching and unified lists.
3. **Agent Integration**: Implemented hooks that call the Claude API for real-time summarization.
4. **Testing**: Automated unit tests for email parsing and priority logic.
